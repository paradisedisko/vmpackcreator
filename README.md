# createvmpackage

## About

Creates Install Anywhere VM packages (.vm) for Windows, Linux and OSX.

## Download


## Usage

Invoking createvmpackage from the command line may look like this:

```bash
createvmpackage <Windows  
```

## Limitations

  * No support for other platforms
